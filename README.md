# Smart Waste Management System

Full-stack application for optimizing urban waste collection using IoT sensors, machine learning, and route optimization.

## Project Structure

```
├── backend/          # FastAPI backend with ML models
└── frontend/         # React dashboard application
```

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- Real-time bin monitoring with IoT sensors
- ML-powered fill-level predictions
- Route optimization for collection trucks
- Citizen complaint portal
- Admin analytics dashboard
- GIS mapping and visualization
